    2. Create `internal/output/printer.go`:
       - Struct `Printer`:
         ```go
         type Printer struct {
             w     io.Writer
             isTTY bool
         }
         ```
       - `func NewPrinter(w io.Writer) *Printer`:
         Detect TTY by checking if `w` implements `interface{ Fd() uintptr }` and calling `term.IsTerminal(int(fd))`. If not castable or not terminal, `isTTY = false`.
         Use `golang.org/x/term` for `IsTerminal` — add to go.mod.
         NOTE: If `golang.org/x/term` adds too much, alternatively check `os.Stdout.Fd()` with lipgloss's built-in renderer. Keep implementation simple.
       - `func (p *Printer) Success(msg string)`: If TTY, print with StyleSuccess. Else print `[OK] {msg}`.
       - `func (p *Printer) Error(msg string)`: If TTY, print with StyleError. Else print `[ERROR] {msg}`.
       - `func (p *Printer) Warning(msg string)`: If TTY, print with StyleWarning. Else print `[WARN] {msg}`.
       - `func (p *Printer) Info(msg string)`: If TTY, print with StyleInfo. Else print `[INFO] {msg}`.
       - `func (p *Printer) Header(msg string)`: If TTY, print with StyleHeader. Else print `=== {msg} ===`.
       - `func (p *Printer) Muted(msg string)`: If TTY, print with StyleMuted. Else print msg as-is.
       - `func (p *Printer) Printf(format string, args ...interface{})`: Plain fmt.Fprintf to writer.
