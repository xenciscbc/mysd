       - Build `idToTask map[int]TaskItem`, `inDegree map[int]int`, `adj map[int][]int`
       - Seed queue with all tasks where inDegree == 0, `sort.Ints(queue)` for determinism
       - BFS layer extraction loop: each iteration = one dependency layer
       - After loop, check `len(processed) != len(tasks)` — if not equal, return `nil, ErrCyclicDependency`

       - Sort tasks within each wave by `task.ID` ascending

    4. `func splitLayer(tasks []TaskItem) [][]TaskItem` — greedy first-fit: for each task, try to place in existing sublayer without file conflict; if none, create new sublayer

    5. `func hasFileConflict(layer []TaskItem, t TaskItem) bool` — exact string match on Files slices. Build `map[string]struct{}` from existing layer files, check if any of `t.Files` exists in the map

    6. `func HasParallelOpportunity(tasks []TaskItem) bool` — returns true if any task has `len(Depends) > 0` or `len(Files) > 0`

    All functions are pure — no I/O, no imports beyond `errors`, `sort`.
