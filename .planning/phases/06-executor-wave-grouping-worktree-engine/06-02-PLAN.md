  <action>
    Create new package `internal/worktree/`.

    **File: `internal/worktree/worktree.go`** (package worktree):

    1. `type WorktreeManager struct`:
       - `RepoRoot string` — absolute path to repo root
       - `WorktreeDir string` — relative dir name, e.g. `.worktrees`
       - `ChangeName string` — current change name for branch prefix

    2. `func (m *WorktreeManager) Create(id int, taskName string) (path string, branch string, err error)`:
       - If `runtime.GOOS == "windows"`, call `m.setLongPaths()`
       - Call `m.CheckDiskSpace(500 * 1024 * 1024)` — 500MB threshold (Claude discretion)
       - Compute: `slug := toSlug(taskName)`
       - Compute: `worktreePath := filepath.Join(m.RepoRoot, m.WorktreeDir, fmt.Sprintf("T%d", id))`
       - Compute: `branchName := fmt.Sprintf("mysd/%s/T%d-%s", m.ChangeName, id, slug)`
       - Execute: `git worktree add -b {branchName} {worktreePath}` with `cmd.Dir = m.RepoRoot`
       - Return path, branch, nil on success

    3. `func (m *WorktreeManager) Remove(id int, branch string) error`:
       - Compute: `worktreePath := filepath.Join(m.RepoRoot, m.WorktreeDir, fmt.Sprintf("T%d", id))`
       - Execute: `git worktree remove --force {worktreePath}` (--force to handle untracked files, per Pitfall 2)
       - Execute: `git branch -d {branch}` — non-fatal on failure (log warning to stderr)
       - Execute: `git worktree prune` — cleanup stale metadata
       - Return nil

    4. `func (m *WorktreeManager) CheckDiskSpace(minBytes uint64) error`:
       - Call `getAvailableBytes(m.RepoRoot)` (platform-specific)
       - If error, return nil (non-fatal — can't determine space)
       - If `available < minBytes`, return error: `"insufficient disk space: need at least %dMB, have %dMB available at %s"`
       - Otherwise return nil

    5. `func (m *WorktreeManager) setLongPaths() error`:
       - Execute: `git config core.longpaths true` with `cmd.Dir = m.RepoRoot`

    6. `func toSlug(name string) string`:
       - Lowercase, replace non-alphanumeric with `-`, collapse multiple `-`, trim leading/trailing `-`
       - Use `strings.Map` + `strings.ToLower` (no regex needed per RESEARCH.md)

    **File: `internal/worktree/diskspace_unix.go`** (`//go:build !windows`):
    - `func getAvailableBytes(path string) (uint64, error)` using `syscall.Statfs`
    - Return `stat.Bavail * uint64(stat.Bsize)`

    **File: `internal/worktree/diskspace_windows.go`** (`//go:build windows`):
    - `func getAvailableBytes(path string) (uint64, error)` using `golang.org/x/sys/windows.GetDiskFreeSpaceEx`
    - Return `freeBytesAvailable`

    **Tests** use `t.TempDir()` + `git init` to create isolated test repos. Tests that run `git worktree add` need a real git repo with at least one commit (empty commit is fine).
