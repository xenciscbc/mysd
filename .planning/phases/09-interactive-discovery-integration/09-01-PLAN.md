    ```go
    package spec

    type DeferredNote struct {
        ID        int    `json:"id"`
        Content   string `json:"content"`
        CreatedAt string `json:"created_at"`
    }

    type DeferredStore struct {
        Notes []DeferredNote `json:"notes"`
    }

    func DeferredPath(specDir string) string
    func LoadDeferredStore(specDir string) (DeferredStore, error)
    func SaveDeferredStore(specDir string, store DeferredStore) error
    func (s *DeferredStore) Add(content string) DeferredNote
    func (s *DeferredStore) Delete(id int) bool
    func CountDeferredNotes(specDir string) (int, error)
    ```

    Implementation details:
    - `LoadDeferredStore`: `os.ReadFile` + `json.Unmarshal`. If `os.IsNotExist(err)` return zero-value `DeferredStore{}` (convention-over-config from Phase 1).
    - `SaveDeferredStore`: `json.MarshalIndent(store, "", "  ")` + `os.WriteFile(path, data, 0644)`.
    - `Add`: compute `id = max(existing IDs) + 1` (start from 1 if empty). Set `CreatedAt = time.Now().Format(time.RFC3339)`. Append to `s.Notes`.
    - `Delete`: linear scan, remove by index, return bool.
    - `CountDeferredNotes`: load store, return `len(store.Notes)`.
    - `DeferredPath`: `filepath.Join(specDir, "deferred.json")`.

    REFACTOR: Ensure consistent error wrapping with `fmt.Errorf("deferred: %w", err)` pattern.
