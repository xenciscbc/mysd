    package roadmap

    import (
        "os"

    }

    type ChangeRecord struct {
        Name           string     `yaml:"name"`

        StartedAt      *time.Time `yaml:"started_at,omitempty"`
        CompletedAt    *time.Time `yaml:"completed_at,omitempty"`
        TotalTasks     int        `yaml:"total_tasks"`
        CompletedTasks int        `yaml:"completed_tasks"`
        MustTotal      int        `yaml:"must_total"`
        MustPassed     int        `yaml:"must_passed"`
    }

    `UpdateTracking(specsDir string, ws state.WorkflowState) error`:
    - Derive project root: `filepath.Dir(specsDir)` (handles both `.specs/` and `openspec/`)
    - Roadmap dir: `filepath.Join(projectRoot, ".mysd", "roadmap")`
    - Create roadmap dir with `os.MkdirAll(roadmapDir, 0755)`
    - Tracking path: `filepath.Join(roadmapDir, "tracking.yaml")`
    - Read existing tracking.yaml. If not found, create new TrackingFile with SchemaVersion "1"
    - Find existing ChangeRecord by `ws.ChangeName`. If not found, create new record with StartedAt = now.
    - Update record: Status = string(ws.Phase)
    - If ws.Phase == state.PhaseArchived, set CompletedAt = now
    - Set UpdatedAt = now on TrackingFile
    - Marshal to YAML, write tracking.yaml
    - Call `GenerateMermaid(tf)` and write result to `filepath.Join(roadmapDir, "timeline.md")`

    `ReadTracking(roadmapDir string) (TrackingFile, error)`:
    - Read tracking.yaml from roadmapDir. If os.ErrNotExist, return zero-value TrackingFile (no error). Same pattern as uat.ReadUAT.

    **File: internal/roadmap/mermaid.go**

    `GenerateMermaid(tf TrackingFile) string`:
    - Use `text/template` (stdlib) with template string:
      ```
      gantt
          title Roadmap
          dateFormat YYYY-MM-DD
          {{range .Changes}}
          section {{.Name}}
              {{.Name}} :{{ganttStatus .}}, {{formatStartDate .}}, {{formatEndDate .}}
          {{end}}
      ```
    - ganttStatus mapping: archived/verified -> "done", executed -> "active", all others -> no modifier
    - formatStartDate: if StartedAt is nil, use UpdatedAt; format as "2006-01-02"
    - formatEndDate: if CompletedAt is nil, use current date or a future placeholder; format as "2006-01-02"
    - Register template functions via `template.FuncMap`
    - Return rendered string. If no changes, return just the gantt header.
