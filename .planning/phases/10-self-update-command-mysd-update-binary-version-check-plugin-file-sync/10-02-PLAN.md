    **Types:**
    ```go
    // PluginManifest tracks official plugin files shipped with a release.
    type PluginManifest struct {
        Version  string   `json:"version"`
        Commands []string `json:"commands"` // filenames like "mysd-update.md"
        Agents   []string `json:"agents"`   // filenames like "mysd-executor.md"
    }

    // ManifestDiff describes what sync operations are needed.
    type ManifestDiff struct {
        AddCommands    []string // files to add to .claude/commands/
        UpdateCommands []string // files to overwrite in .claude/commands/
        DeleteCommands []string // files to remove from .claude/commands/
        AddAgents      []string
        UpdateAgents   []string
        DeleteAgents   []string
    }
    ```

    **Functions:**
    - `LoadManifest(path string) (*PluginManifest, error)` — reads JSON file. Returns `(nil, nil)` if file does not exist (convention-over-config, same pattern as deferred.go). This nil represents "pre-v1.1 installation without manifest" (D-17).
    - `SaveManifest(path string, m *PluginManifest) error` — writes indented JSON.
    - `DiffManifests(old, new *PluginManifest) ManifestDiff` — three-way comparison:
      - If `old == nil`: all new.Commands are "add", all new.Agents are "add", zero deletes (D-17 backward compat).
      - If both non-nil: compute add (in new, not in old), update (in both), delete (in old, not in new).
    - `GenerateManifest(pluginDir, version string) (*PluginManifest, error)` — scans `{pluginDir}/commands/*.md` and `{pluginDir}/agents/*.md`, returns manifest with filenames (not full paths). Excludes CLAUDE.md files.

    Tests: standard table-driven tests for DiffManifests edge cases, temp dir tests for Load/Save/Generate.

    // SyncPlugins applies ManifestDiff by copying files from sourceDir to targetDir.

    Implementation:
