    - ParseChecksumFile returns error for filename not found in checksums.txt
    - ApplyUpdate on temp dir creates new binary and removes .old file
    - ApplyUpdate with checksum mismatch returns error and does not replace binary
    - replaceExecutable on Windows renames old to .old then writes new (build tag: windows)
    - replaceExecutable on Unix writes new with 0755 permissions (build tag: !windows)
    - Rollback restores .old to original path
  </behavior>
  <action>
    **selfupdate.go** (no build tags, shared logic):
    ```go
    // DownloadFile downloads a URL to a temp file, returns path.
    func DownloadFile(ctx context.Context, url string, httpClient *http.Client) (string, error)

    // VerifyChecksum compares SHA256 of filePath against expected hex string.
    func VerifyChecksum(filePath, expectedHex string) error

    // ParseChecksumFile parses GoReleaser checksums.txt format:
    // "{hash}  {filename}\n" — double space separator.
    // Returns the hash for the given filename.
    func ParseChecksumFile(content, filename string) (string, error)

    // ExtractBinary extracts the mysd binary from tar.gz or zip archive.
    // Returns path to extracted binary in a temp directory.
    func ExtractBinary(archivePath string, isZip bool) (string, error)

    // ApplyUpdate orchestrates the full update:
    // 1. Download archive + checksums.txt
    // 2. Verify archive checksum
    // 3. Extract binary from archive
    // 4. Call replaceExecutable (platform-specific)
    // 5. On any failure after step 4, call Rollback
    func ApplyUpdate(ctx context.Context, httpClient *http.Client, release ReleaseInfo, currentExePath string) error

    // Rollback restores {exePath}.old back to exePath.
    func Rollback(exePath string) error
    ```

    **selfupdate_windows.go** (build tag `//go:build windows`):
    ```go
    // replaceExecutable implements Windows rename-then-replace (D-09):
    // 1. Rename current exe to exe + ".old"
    // 2. Rename/move new binary to exe path
    // 3. Attempt to delete .old (may fail if still locked — that's OK)
    func replaceExecutable(currentExe, newBinaryPath string) error
    ```

    **selfupdate_unix.go** (build tag `//go:build !windows`):
    ```go
    // replaceExecutable on Unix:
    // 1. Rename current exe to exe + ".old"
    // 2. Copy new binary to exe path with 0755 permissions (D-12)
    // 3. Remove .old file
    func replaceExecutable(currentExe, newBinaryPath string) error
    ```

    For tar.gz extraction: use `archive/tar` + `compress/gzip` from stdlib.
    For zip extraction: use `archive/zip` from stdlib.

    SHA256 checksum: use `crypto/sha256` from stdlib. Compare hex-encoded hash.
