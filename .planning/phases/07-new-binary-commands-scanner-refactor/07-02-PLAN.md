    **COMPLETE REPLACEMENT of scanner.go** (per D-02 — no backward compat with PackageInfo).

    New structs in `internal/scanner/scanner.go`:

    ```go
    type ScanContext struct {
        RootDir         string         `json:"root_dir"`
        PrimaryLanguage string         `json:"primary_language"` // "go" | "nodejs" | "python" | "unknown"
        Files           map[string]int `json:"files"`            // extension -> count, e.g. {".go": 42}
        Modules         []ModuleInfo   `json:"modules"`          // language-agnostic module list
        ExistingSpecs   []string       `json:"existing_specs"`
        ExcludedDirs    []string       `json:"excluded_dirs"`
        TotalFiles      int            `json:"total_files"`
        ConfigExists    bool           `json:"config_exists"`    // openspec/config.yaml exists?
    }

    type ModuleInfo struct {
        Name string `json:"name"` // module/package name
        Dir  string `json:"dir"`  // relative path
    }
    ```

    New `detectPrimaryLanguage(root string) string` function:
    - Check file markers in order (per D-01): go.mod -> "go", package.json -> "nodejs", requirements.txt -> "python", pyproject.toml -> "python"
    - If none match, return "unknown"
    - Use `os.Stat(filepath.Join(root, marker))` for each

    Rewrite `BuildScanContext(root string, exclude []string) (ScanContext, error)`:
    - Call `detectPrimaryLanguage(root)` for primary_language
    - Walk directory tree (same hidden dir + exclude logic as current implementation)
    - Count ALL files by extension into `Files` map (not just .go files — per D-03 binary collects all metadata)
    - Detect modules: For Go: each directory with .go files is a module. For Node.js: each directory with index.js/index.ts or containing package.json. For Python: each directory with __init__.py. For unknown: each top-level directory with source files.
    - Check `openspec/config.yaml` existence via `spec.ReadOpenSpecConfig(root)` — if Locale is non-empty, config_exists=true. Or simpler: `os.Stat(filepath.Join(root, "openspec", "config.yaml"))` for config_exists.
    - Detect existing specs: check for `openspec/specs/` or `.specs/changes/` directories (use `spec.DetectSpecDir` then walk changes dir)
    - REMOVE PackageInfo struct entirely — no backward compat (D-02)

    IMPORTANT: `Files` map must use lowercase extension keys with leading dot (e.g., ".go", ".js", ".py").
    IMPORTANT: `Modules` must have empty slice `[]` not nil when no modules found (for clean JSON serialization).
    IMPORTANT: `ExcludedDirs` must have empty slice `[]` not nil.

    **Replace all tests** in scanner_test.go. The old tests reference `ctx.Packages` and `PackageInfo` — they must be completely rewritten to test the new struct fields.
